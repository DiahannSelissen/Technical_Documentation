Wat is Git

Git is een versiebeheersysteem waarmee je wijzigingen in bestanden kunt bijhouden. Het wordt veel gebruikt in softwareontwikkeling om samen te werken aan code.

Met Git kun je teruggaan naar vorige versies, nieuwe versies maken en samenwerken zonder elkaars werk te overschrijven. Gebruik git init om een repository te starten.

Eerste Commit maken

Een commit is een momentopname van je project. Het bevat de wijzigingen die je hebt gemaakt sinds je laatste commit.

Voordat je kunt committen, moet je Git initialiseren in je map met git init.

Daarna voeg je bestanden toe met git add . en commit je ze met git commit -m "Eerste commit".

Branching en Merging

Branches zijn aparte lijnen van ontwikkeling. Ze maken het mogelijk om nieuwe functies te bouwen zonder de hoofdversie te verstoren.

Je kunt een nieuwe branch maken met git branch nieuwe-feature.

En ernaar overstappen met git checkout nieuwe-feature.

Als je klaar bent, kun je je branch mergen met git merge nieuwe-feature.

Remote repositories en GitHub

Remotes zijn online versies van je project, bijvoorbeeld op GitHub.

Je koppelt je lokale repo aan een remote met git remote add origin https://github.com/jouwwaarde/projectnaam.git.

Daarna push je je commits online met git push -u origin main.

Veelgebruikte commando
  • git status – toont de status van je bestanden
  • git log – laat commitgeschiedenis zien
  • git diff – toont veranderingen tussen versies
  • git clone [url] – kloont een remote repo
  • git pull – haalt en merge't updates van remote

Met deze commando’s kun je snel de meeste taken in Git uitvoeren.
